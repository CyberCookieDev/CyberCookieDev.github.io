About HandyTools

This project is the all-in-one solution to storing and managing your Python scripts. Now has even more features than in the beta.

Easy and straightforward.

Everything works out of the box - there are even a few scripts included. If you want to add your own script, just use the add function, or follow the instructions.txt in the scripts folder.

  • No spyware collecting your data - everything runs locally.
  • Completely free and open-source - if you want you can look through all the code.
  • Includes Developer Tools that help you improve imported scripts and view their text in terminal. If you want to learn more, please read the docs.
