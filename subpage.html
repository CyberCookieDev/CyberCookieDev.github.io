Get Started with HandyTools

Follow these simple steps to install and use the project on your system.

Installation Guide

  1. Step 1: Clone the GitHub repository: 
    git clone https://github.com/CyberCookieDev/handytools.git
  2. Step 2: Navigate to the project folder: 
    cd cd handytools
  3. Step 3: Install dependencies (ensure you have Python and pip installed): 
    pip install -r requirements.txt
  4. Step 4: Navigate to internal code directory and un the application: 
    main_code.py
Open on Github

Project Requirements

  • Python 3.6+
  • pip (Python package manager)
  • colorama~=0.4.6
  • Send2Trash~=1.8.3
  • rich~=13.8.0
  • psutil~=6.0.0
  • screeninfo~=0.8.1
  • black~=24.10.0
  • setuptools~=68.2.0
  • Pygments~=2.19.1
